## Swiggy Sales & Customer Insights Dashboard

## Project Overview

This project focuses on analyzing Swiggy sales data to understand customer ordering behavior, sales performance, and business trends. The objective was to transform raw transactional data into an interactive Excel dashboard that helps users explore key business metrics and gain meaningful insights for decision-making.

The project covers the complete analytics workflow, including data cleaning, data analysis, dashboard creation, and visualization using Microsoft Excel.

## Objective

- Analyze overall sales performance.
- Understand customer ordering trends.
- Compare Veg and Non-Veg sales.
- Identify high-performing cities and states.
- Evaluate monthly, weekly, and quarterly sales trends.
- Build an interactive dashboard for dynamic analysis.

## Project Workflow

### Data Cleaning
- Removed inconsistencies and duplicate records.
- Corrected formatting issues.
- Prepared the dataset for analysis.

### Data Analysis
Performed analysis using Pivot Tables and Excel functions to study:
- Monthly sales trends
- Weekly sales trends
- Food category performance
- Restaurant performance
- State-wise sales
- Quarterly sales performance
- Customer ratings and order statistics

### Dashboard Development
Created an interactive dashboard using:
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Excel Formatting and Dashboard Design

## Dashboard Highlights

- Total Sales
- Total Orders
- Average Order Value
- Average Customer Rating
- Monthly Sales Trend
- Weekly Sales Trend
- Day-wise Sales Trend
- Sales by Food Type
- Quarterly Performance Summary
- Sales by State
- Top Performing Cities

## Key Insights

- Monthly and weekly sales trends help identify peak ordering periods.
- Veg and Non-Veg sales distribution provides insights into customer food preferences.
- State-wise sales analysis highlights the strongest performing markets.
- Quarterly performance helps compare business growth throughout the year.
- KPI cards provide an overall snapshot of business performance, including total sales, orders, average order value, and customer ratings.
